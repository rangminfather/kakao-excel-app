카톡보고정리

매일 반복되는 보고 정리를 조금 덜기 위해 만든 개인용 도구입니다.
설치에 앞서, 아래 세 가지만 살펴봐 주세요.

── 외부로 나가는 것
"정형화" 실행 시, 선택된 카카오톡 대화 본문은 Google Gemini API 서버로 전송되어
정돈된 표의 형태로 돌아옵니다. 이 경로 외에는 외부로 흐르는 정보가 없습니다.

── 이 PC에 머무는 것
입력하신 Gemini API 키는 Windows 자격 증명 관리자로 암호화되어 이 기기에만 보관됩니다.
제작자의 서버나 어떠한 외부 저장소도 경유하지 않습니다.

── 책임의 한계
본 소프트웨어는 별도의 보증 없이 "있는 그대로" 제공됩니다. 사용 중에 생길 수 있는
어떠한 손실—데이터, API 비용, 결과 오차, 업무 지연—에 대해서도 제작자는
법적 책임을 지지 않습니다. 다만 불편하거나 어긋난 점을 발견하시면,
언제든 알려주세요. 기꺼이 고쳐 드리겠습니다.

동의하신다면, "동의함"을 눌러 설치를 이어가 주세요.
